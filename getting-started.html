# Getting Started

# Installation

First step is to install it using yarn or npm:

yarn add verte

# or use npm
npm install verte --save

# Basic usage

<template>
  <verte :picker="wheel" :model="rgb"></verte></template>

<script>
  import Verte from './dist/verte.js';
  // register component globally
  Vue.component(Verte.name, Verte);
  new Vue ({
    el: '#app',
    // OR register locally
    components: { Verte }  });
</script>
